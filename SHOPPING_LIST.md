## ❌ DO NOT BUY

| Item | Why Not |
|------|---------|
| Windows license | You run Linux. Save €100-150. |
| RGB fans / lighting | Adds cost, no performance benefit |
| AIO liquid cooler | More complex, leak risk, AK620 is fine |
| Thermal paste from unknown brand | Stick to Arctic, Noctua, Thermal Grizzly |
| Cheap PSU (&lt;80+ Bronze) | Can fry your entire build |
| Used GPU (unless you accept risk) | Stick with new 4070 Ti S |
| DDR4 RAM | Wrong platform — need DDR5 for AM5 |
| B550 motherboard | Wrong socket — need B650/X670 for AM5 |
