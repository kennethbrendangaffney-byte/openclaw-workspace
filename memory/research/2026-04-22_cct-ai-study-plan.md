## 📊 Assessment Strategy

### General Approach
- **Week 3-4 of each module:** Start assignment prep (don't wait until deadline)
- **Document everything:** GitHub commits, notes, screenshots
- **Ethics module:** Keep a running journal of real-world AI ethics cases
- **Programming:** Code daily, even if just 30 minutes
- **Business modules:** Connect theory to your own experience

### Portfolio Building
- Maintain one GitHub repo for the entire course
- Each module gets a folder with code + README
- Final portfolio = evidence of progression
- Include your capstone project prominently
