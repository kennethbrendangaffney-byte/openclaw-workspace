## Executive Summary

The AI tooling landscape in 2026 is defined by three major shifts:
1. **Agent-first interfaces** - Tools now orchestrate AI fleets rather than single completions
2. **Local/self-hosted maturity** - Consumer hardware can now run serious models (8B-13B parameters)
3. **Performance gap closure** - Frontier models are within reach of each other; choice depends on use case and budget

## 🖥️ Development & Coding Tools

### Cursor 3 (April 2026)
- **What it is:** Agent-first IDE rebuilt from scratch under codename "Glass"
- **Key feature:** Up to 8 parallel AI agents working simultaneously ("Agent Fleets")
- **New model:** Composer 2 - frontier-level coding intelligence at $0.50/M input, $2.50/M output
- **Benchmarks:** 61.3 on CursorBench, 61.7 on Terminal-Bench 2.0, 73.7 on SWE-bench Multilingual
- **Verdict:** If you prefer visual editors over terminal workflows, this is the current leader
- **Link:** https://cursor.com

### Claude Code (Anthropic)
- **What it is:** Terminal-based coding agent, now powering GitHub Copilot Enterprise
- **Key feature:** Multi-agent coordination - splits large tasks into parallel subtasks
- **Context:** Works with your entire codebase, plans changes across multiple files, writes code, runs tests
- **Best for:** Developers comfortable in terminal who want deep project understanding
- **Available through:** Claude subscription / Anthropic API

### Claude 4.6 (Anthropic)
- **Key feature:** 1-million-token context window
- **Use case:** Feed it entire codebases, legal contracts, months of meeting transcripts
- **Best for:** Large-document analysis, long-form writing, complex coding tasks
- **Price:** $20/month (Claude Pro) or API

### GPT-5.4 (OpenAI)
- **Key feature:** Native "computer use" - autonomously navigates software, clicks buttons, fills forms
- **Use case:** Repetitive tasks spanning multiple applications (pull data from one app, transform it, enter into another)
- **Available through:** ChatGPT Plus/Pro

### Gemini 3.1 Pro (Google)
- **Key feature:** 13 of 16 benchmark leads in Q1 2026 at ~1/3 the cost of competitors
- **Integration:** Deep Google Workspace integration (Docs, Sheets, Slides, Gmail, Calendar)
- **Standout:** 70% success rate on complex spreadsheet generation from natural language
- **Best for:** Teams already in Google Workspace ecosystem

## 🏠 Local & Self-Hosted Options

### LocalAI
- **What it is:** Complete open-source AI stack alternative to OpenAI/Anthropic
- **Features:** LLM inference, autonomous agents, document intelligence, RAG
- **Philosophy:** No cloud, no limits, no compromise
- **Link:** https://localai.io

### DreamServer (Light-Heart-Labs)
- **What it is:** Local AI server - LLM inference, chat UI, voice, agents, workflows, RAG, image generation
- **Tech stack:** Rust (33.9%), Python
- **GitHub:** 441 stars, 143 forks
- **Pitch:** "Local AI anywhere, for everyone — no cloud, no subscriptions"

### Ollama vs LM Studio vs vLLM (2026 Landscape)
- **Ollama:** CLI-focused, simplicity, API integration for developers. 162,000+ GitHub stars
- **LM Studio:** GUI-focused, beginner-friendly desktop app. 5x memory efficiency improvements in 2026
- **vLLM:** Production serving, high-throughput inference
- **Jan:** Newcomer with different philosophy - open-source alternative to both

### Practical Local LLM Guide (2026)
- Enterprise LLM spending doubled to $8.4 billion in 2025
- 72% of companies plan to increase AI budgets further
- Key consideration: Every API call sends your data somewhere else
- Local options now viable for: 7B-13B parameter models on consumer hardware (24GB RAM + decent GPU)

### Fyxer AI Framework
- Three-part test for evaluating AI tools:
  1. Does it handle a specific, high-frequency task?
  2. Does it fit inside software you already use?
  3. Can you see workflow change within two weeks?

### Emerging Patterns
- **Model-swapping interfaces:** Tools that let you switch between Claude, GPT, Gemini based on task
- **Agent orchestration:** Single tools managing multiple AI agents in parallel
- **Embedded AI:** AI features built directly into existing workflows (Slack, Notion, Gmail) rather than standalone apps

## 💡 Key Insights for Local/Self-Hosted Setups

### Hardware Reality Check (2026)
- **8GB VRAM:** Entry-level video generation (Wan 1.3B, AnimateDiff)
- **12-16GB VRAM:** Mid-tier creative work (LTX-Video, HunyuanVideo v1.5 with offload)
- **24GB+ VRAM:** Serious local LLM work (13B models, full quality video)
- **60GB+ VRAM:** Research/professional video (Mochi 1, Open-Sora 2.0)

### Cost Considerations
- Cloud API costs: $8.4B industry spending in 2025, doubling trend continues
- Local setup: One-time hardware cost vs ongoing API subscriptions
- Hybrid approach: Local for sensitive/personal work, cloud for heavy lifting

### Privacy Angle
- "Every API call sends your data somewhere else"
- Local options now competitive in quality for many use cases
- Self-hosted RAG (Retrieval-Augmented Generation) viable for document analysis
