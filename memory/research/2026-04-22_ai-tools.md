## 📊 Key Trends (April 2026)

1. **Open-source = competitive.** The best open models now match or beat proprietary on coding/math benchmarks. GLM-5.1 (MIT license) is #1 on SWE-Bench Pro.

2. **MoE dominates.** Mixture-of-Experts architectures let trillion-parameter models run on consumer hardware. Active parameter count matters more than total.

3. **Agentic AI is here.** Models now autonomously plan, execute, test, and iterate. GLM-5.1 ran 6,000+ tool calls in a single 8-hour session.

4. **MCP standardization.** Model Context Protocol is becoming the universal integration layer for AI agents and tools.

5. **Self-hosting is practical.** With Qwen 3.6 (3B active params) running on dual RTX 5060 Ti, frontier-quality coding is achievable on consumer hardware.

6. **Video market reshuffled.** OpenAI shut down Sora (March 2026), redirecting to robotics. Runway, Kling, and Google Veo now lead.

7. **Licensing matters.** MIT/Apache 2.0 = unrestricted. Meta Llama blocks >700M MAU. DeepSeek V4 has custom restrictions.
