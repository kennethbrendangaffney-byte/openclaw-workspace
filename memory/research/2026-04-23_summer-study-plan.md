## Phase 1: Python Foundations (Weeks 1–6)

### Week 1–2: The Basics
- Installing Python, running code
- Variables, types (`int`, `float`, `str`, `bool`)
- `print()`, `input()`, basic math
- Conditionals: `if / elif / else`
- **Mini-project:** Tip calculator, simple quiz game

### Week 3–4: Logic & Loops
- `for` loops (iterate over lists, ranges)
- `while` loops (repeat until condition met)
- Lists and dictionaries (store data)
- Basic file reading/writing
- **Mini-project:** To-do list app, file organizer

### Week 5–6: Functions & Modules
- Writing your own functions (`def`)
- Parameters, return values
- Using libraries (`import`)
- Error handling (`try / except`)
- **Mini-project:** Password checker, simple calculator with functions

**Resources:**
- "Python for Everybody" — Dr. Chuck (YouTube, free)
- "Automate the Boring Stuff" — Al Sweigart (free online)
- Practice: [python.org](https://docs.python.org/3/tutorial/) official tutorial

## Phase 2: Python Practice (Weeks 7–12)

### Week 7–8: Working with Data
- **NumPy** — arrays, math operations
- **Pandas** — DataFrames (like Excel in code)
- Reading CSV files, filtering data
- Basic statistics (mean, median, standard deviation)
- **Mini-project:** Analyze a dataset (your choice — anything interesting)

### Week 9–10: Visualization & APIs
- **Matplotlib** — create graphs, charts
- **Requests** — fetch data from the internet
- JSON handling (API responses)
- Calling Ollama's API from Python
- **Mini-project:** Fetch weather data, plot a chart, or build a simple chatbot

### Week 11–12: Small Projects
- Build something useful for YOU
- Ideas:
  - Daily research file summarizer
  - System health monitor (the cron jobs, but Python-native)
  - KDP image generator (kids' colouring pages)
  - Simple web scraper
- **Goal:** Have 2-3 small projects on your GitHub

**Resources:**
- Google's Machine Learning Crash Course (free)
- [Kaggle Learn](https://www.kaggle.com/learn) — short courses, free
- Ollama API docs: [ollama.com](https://ollama.com)

**Khan Academy — free, self-paced:**

| Month | Topic | Why It Matters |
|-------|-------|--------------|
| May | Algebra review | Foundation for everything |
| June | Statistics & probability | ML is literally applied statistics |
| July | Linear algebra (basics) | How neural networks work under the hood |

## Realistic Pacing Rules

| Rule | Why |
|------|-----|
| **If you miss a day, skip it.** Don't cram. | Consistency > intensity |
| **If a concept doesn't click, park it.** | Come back later — sometimes things need to marinate |
| **30 minutes is better than 0 minutes.** | Short sessions count |
| **Sundays are sacred.** No studying. | Recovery prevents burnout |
| **Work days = no studying.** | Those 10-hour shifts are enough |
| **Projects > exercises.** | Building something real is the best teacher |
