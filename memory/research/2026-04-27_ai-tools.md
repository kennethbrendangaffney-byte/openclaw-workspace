## 📌 Key Trends (Q2 2026)

1. **Reasoning is now the default** — OpenAI retired the o1/o3 branding. GPT-5.4 Thinking, Claude Opus 4.7, and Gemini 3.1 Pro all blend reasoning into the main model.
2. **Agents have become a product category** — Microsoft Agent 365 launches May 1, 2026. Cursor 3 rebuilt around parallel agents. Claude Cowork is GA on macOS/Windows.
3. **Open-source models match or beat proprietary** — GLM-5.1 topped SWE-Bench Pro ahead of GPT-5.4 and Claude Opus 4.6. Gemma 4 runs on consumer hardware. Qwen 3.6 scores 73.4% on SWE-bench Verified with only 3B active params.
4. **Model identity matters less than stack integration** — Copilot Wave 3 uses Claude, GPT, and Microsoft models in the same flow. The differentiator is workflow integration, not which model you pick.
