## RAM Buying Strategy

**Option A: Buy 4×32GB now (€700)**
- All matching sticks from same batch
- Guaranteed compatibility
- One purchase, done

**Option B: Buy 2×32GB now (€350), add 2×32GB later**
- Spread cost over time
- Test the build first
- Risk: price fluctuations, availability changes
- **Note:** All 4 slots filled later = no easy path to 256GB

## What Changes vs 64GB Build

| Aspect | 64GB | 128GB |
|--------|:----:|:-----:|
| Slots filled | 2 of 4 | 4 of 4 |
| Upgrade to 256GB | Easy (add 2 sticks) | Hard (replace all 4) |
| Total cost | €2,020 | €2,370 |
| Budget overrun | +€20 | +€370 |

## My Take

**You made the right call for YOU.** Here's why:

- €350 over 4-5 years = €70/year for never worrying about RAM
- You explicitly wanted headroom — now you have maximum headroom
- By the time DDR5 is cheap enough to make 128GB "obvious," you'll already have it
- Your use case might grow (business automation, bigger projects)

**The compromise:** All 4 RAM slots filled means 256GB later requires replacing all sticks. But honestly, if you need 256GB, you'll probably be buying a whole new machine by then anyway.
