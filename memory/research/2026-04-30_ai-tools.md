- **ElevenLabs Image & Video (Beta)** — Expands from audio to full creative platform. Generate images/video with Veo/Sora/Kling, add voices/music/sfx, lipsync. End-to-end creative production.

### Infrastructure & Standards

- **Anthropic MCP** — 97 million installs. Now foundational infrastructure for AI agents. Every major provider ships MCP-compatible tooling.
- **Google TurboQuant** — KV cache compression reducing inference memory 6x with no accuracy loss. Immediate market impact (memory chip stocks dropped).
- **Anthropic ends third-party Claude subscriptions** (April 4) — OAuth tokens no longer work in external tools like OpenClaw. Must use pay-as-you-go API keys now.

## 🏠 Local / Self-Hosted Stack Recommendations

### Minimal Setup (Your Current Hardware — 24GB RAM)

```
Runtime:    Ollama
Chat UI:    Open WebUI or terminal
Coding:     Continue.dev (VS Code) + Ollama backend
Models:     qwen3.5:4b (fast), llama3.1:8b (quality), gemma4:27b (new, test carefully)
Embeddings: nomic-embed-text (for RAG/memory)
```

### Expanded Self-Hosted Stack

```
LLM Runtime:     Ollama / vLLM / llama.cpp
Chat Interface:  Open WebUI / LibreChat / Lobe Chat
Coding:          Continue.dev / Aider / Tabby
Automation:      n8n + local LLM nodes
Image:           ComfyUI + Stable Diffusion
Voice:           Piper TTS (local) / Coqui TTS
Transcription:   Cohere Transcribe (open source) / Whisper (local)
Vector DB:       Chroma / Qdrant / pgvector (for RAG)
```

### Key Local Tools to Explore

- **Open WebUI** — Feature-rich chat interface for Ollama. Supports RAG, multi-user, model management.
- **LibreChat** — Open source ChatGPT clone with multi-model support.
- **Lobe Chat** — Modern UI, plugin system, TTS/STT.
- **n8n** — Self-hosted workflow automation with AI nodes. Zapier alternative.
- **ComfyUI** — Node-based Stable Diffusion interface. Unlimited local image generation.
- **Immich** — Self-hosted photo/video backup with AI tagging (face recognition, object detection).
- **Paperless-ngx** — Document management with OCR and AI classification.

## 📊 Trends to Watch

1. **Agentic shift** — 2026 is the year AI moves from chat to autonomous action. Every major platform is adding agent capabilities.
2. **Open models catching up** — Gemma 4, Llama 4, Qwen 3.5, DeepSeek — the gap between open and proprietary is narrowing fast.
3. **Platform consolidation** — Tools becoming full platforms (ElevenLabs adding image/video, Slack becoming AI work layer).
4. **Memory/efficiency breakthroughs** — TurboQuant and similar innovations making long-context cheaper.
5. **MCP as standard** — Model Context Protocol is now the default for agent-tool integration.
6. **Third-party subscription crackdown** — Anthropic's move may signal broader restrictions on OAuth-based tool access.
