## Kimi vs Moonshot API Keys (23:06)

**Discovery:** Kimi Coding API key (`api.kimi.com/coding`) does NOT work on Moonshot platform (`api.moonshot.ai`). They are completely separate services.

- Kimi Coding key format: `sk-kimi-...` for `api.kimi.com/coding`
- Moonshot key: Different format for `api.moonshot.ai/v1/chat/completions`

**Web search provider switched to DuckDuckGo** — zero auth, completely free, working immediately.
