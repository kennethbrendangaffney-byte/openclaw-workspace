### Key Findings:
1. **RTX 4080 Super — SKIP:** Same 16GB VRAM as 4070 Ti Super but €300 more. 43% more cost for 15% more speed. BestGPUforAI.com calls it "a bad buy for AI."

2. **Used RTX 3090 24GB — BEST VALUE:**
   - eBay pricing: €650-800
   - Complete build: €1,730 (€180 cheaper than 4070 Ti Super build)
   - 24GB VRAM = 70B models fit entirely on GPU at 5-8 tok/s
   - Risk: used/mining wear. Buy from sellers with 500+ feedback + return policy

3. **Dual RTX 3090 — 48GB MONSTER:**
   - Cost: €2,300-2,600
   - llama.cpp --tensor-split distributes layers across both cards
   - 70B fits entirely, 8-12 tok/s, no CPU offloading
   - Needs 1000W PSU, more heat/noise, ~70-80% scaling efficiency

4. **RTX 50 Series (Blackwell) — DON'T WAIT:**
   - Only RTX 5090 (32GB) changes the game at €2,500+ (sold out)
   - 5070 Ti/5080 still have 16GB wall — same as 4070 Ti Super
   - Newer but not transformative for LLMs

5. **AMD RX 7900 XTX — SKIP:**
   - 24GB VRAM, ~€950, ROCm support "less seamless"
   - 20-30% slower than RTX 3090 in llama.cpp
   - Only for AMD fans or existing owners
